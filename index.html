Portfolio

Gruvi

A game inspired by Simon. Users can play in 'regular mode', exercising visual/aural memory, or 'expert mode', exercising sense of pitch. Built using Javascript/jQuery, HTML5 Web Audio API, Pusher and Ruby-on-Rails.

CycleSmart

NYC bike app that renders recent bike-related accidents, bike shops, Citibike stations and near-by Groupon deals onto a Google map. Users can also save bike routes & directions in a 'My Routes' list. Built using Ruby-on-Rails, Google Maps API and Javascript/jQuery.

Reelpix

Users can browse & search movie/TV titles via Crackle and create public playlists that link to the titles. Built using Ruby-on-Rails and Crackle's API.
