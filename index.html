Gruvi

A game inspired by Simon. Users can play in 'regular mode' (visual/aural memory), or 'expert mode' (sense of pitch). Built using Javascript/jQuery, HTML5 Web Audio API, Pusher and Ruby on Rails. Group project built during General Assembly's WDI program.

Reelpix

Users can browse & search movie/TV titles via Crackle and create public favorites lists that link to the titles. Built using Ruby on Rails and Crackle's API. Individual project built during General Assembly's WDI program.

CycleSmart

NYC bike app that renders recent bike-related accidents, bike shops, Citibike stations and near-by Groupon deals onto a Google map. Users can also save bike routes & directions in a 'My Routes' list. Built using Ruby on Rails, Google Maps API and Javascript/jQuery. Group project built during General Assembly's WDI program.

About Me

Michael built his first webpage around the age of 12. Over a decade later, he decided to take up building for the internet once again. In 2014, he completed General Assembly's Web Development Immersive program and has since been developing professionally. When working, Michael enjoys coffee, Spotify and NPR. When not working, he enjoys good beer & playing his guitar. He currently resides in Brooklyn, New York.
